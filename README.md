Сервис упакован в Docker. Для запуска достаточно находиться в директории с файлом _'manage.py'_ и в терминале ввести следующее:
```shell
    docker-compose build
    docker-compose up
```

Доступ к веб сервису:
```shell
    http://127.0.0.1:8000/ - в браузере
```
